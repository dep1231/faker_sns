# アプリケーション概要

シンプルな機能を持った SNS サイト

# アプリケーション機能、技術一覧

・認証機能<br>
・テキスト投稿機能<br>
・画像投稿機能<br>
・投稿にコメントをつける機能<br>
・フォロー機能<br>
・いいね機能<br>
・プロフィール編集機能<br>
・アバター、カバー画像変更機能

# アプリケーション技術

MongoDB、Express、React、NodeJs を使用して構築しています。
