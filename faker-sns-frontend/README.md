# アプリケーション概要

シンプルな機能を持った SNS サイト

# アプリケーション機能、技術一覧

・認証機能
・テキスト投稿機能
・画像投稿機能
・投稿にコメントをつける機能
・フォロー機能
・いいね機能
・プロフィール編集機能
・アバター、カバー画像変更機能

# アプリケーション技術

MongoDB、Express、React、NodeJs を使用して構築しています。
